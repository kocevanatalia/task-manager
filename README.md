# 🌸 My Tasks 🌸

A pastel-themed task manager web app built with HTML, CSS, and JavaScript.

## ✨ Features
- Add new tasks
- Mark tasks as completed
- Edit tasks inline (no popups)
- Delete tasks
- Add optional notes for each task
- Save tasks using localStorage (data persists after refresh)
- Keyboard support (press Enter to add or edit tasks)
- Aesthetic pastel UI design with emoji-based controls
- Dark Mode Version
- Drag & Drop task reordering

## 🛠️ Technologies Used
- HTML
- CSS
- JavaScript
- localStorage

## 🎨 UI Highlights
- Pastel color palette (pink, purple, soft tones)
- Rounded, modern design
- Custom typography (Google Fonts)
- Subtle hover animations
- Dashed “cute” task borders
- Icon-based buttons for cleaner UI
- Inline editing for a smoother user experience

- DOM manipulation with JavaScript
- Managing UI state (editing mode)
- Handling user interactions (click, keyboard events)
- Using localStorage for persistent data
- Improving UI/UX with custom styling and design decisions
